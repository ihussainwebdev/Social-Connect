# Social Connect App

WebView app for socialconnect.wuaze.com

## APK Kaise Banaye (FREE - Bina Android Studio)

### Step 1: GitHub Account
- github.com pe jao
- Sign up karo (free hai)

### Step 2: Repository Banao
- "New Repository" click karo
- Name: SocialConnect
- Public rakho
- Create karo

### Step 3: Files Upload Karo
- "uploading an existing file" click karo
- SocialConnect folder ki SAARI files drag karo
- "Commit changes" karo

### Step 4: Actions Enable Karo
- "Actions" tab pe jao
- "I understand my workflows" click karo
- Green button click karo

### Step 5: APK Download Karo
- Actions tab mein build complete hone do (2-3 min)
- Green checkmark aaye to click karo
- "SocialConnect-APK" download karo
- Unzip karo - app-debug.apk milegi!

## Features
- socialconnect.wuaze.com load hoti hai
- Pyara No Internet page
- Splash screen
- Pull to refresh
- Back button support
